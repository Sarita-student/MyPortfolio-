# Sarita Kumari - Portfolio Website

A modern personal portfolio built with **React** and **Tailwind CSS**, inspired by a dark-themed developer portfolio layout with a fixed sidebar navigation.

## Features

- **Hero Section** – Gradient background with developer illustration and typing animation
- **About Me** – Professional summary from your CV
- **Skills** – Categorized tech stack grid
- **Projects** – Carousel showcase with GitHub & live demo links
- **Resume** – Accordion with Experience, Education & Achievements
- **Contact** – Form that opens your email client

## Tech Stack

- React 19 + Vite
- Tailwind CSS 4
- React Icons

## Getting Started

```bash
npm install --legacy-peer-deps
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build for Production

```bash
npm run build
npm run preview
```

## Deploy

Deploy the `dist` folder to [Vercel](https://vercel.com), Netlify, or GitHub Pages.

## Customize

Edit your info in `src/data/portfolioData.js` — name, skills, projects, education, and contact details.
